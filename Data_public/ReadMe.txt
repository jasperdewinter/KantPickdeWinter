See the paper for an overview of the names and non-publlic & public data. See the Excel-file in "DFROG datamodel mrt2019 (DFROG).xlsx" for an overview of the non-publlic data and public data. See Table A.2 in the paper. 

Regular_public: the data in the paper but withot all Refinitiv and ECB restricted data
